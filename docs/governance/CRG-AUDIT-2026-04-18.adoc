=== Ecosystem-overlap alarm (Grade D+ honesty check)

with AGPL-3.0 header, private-visibility badge, and a substantial

`CLADE.a2ml` that declares:

- `parent = "Proof-of-work verification tooling"` — not a real parent object
- `lineage.type = "standalone"`

["org.hyperpolymath.boj-server"]` and `clade: "unknown"`. The two
clade/anchor files disagree about parentage, and neither matches the
README's self-presentation as a Bevy + Z3 puzzle game.

This is not itself a CRG-reducing finding, but it is an
*identity-and-scope* finding. A repo that cannot state in one file who
its parent is, and where a sister repo is the actual game research,
cannot honestly claim "C — Alpha-stable, home-stable, dogfooded" —
because "dogfooded as what, where" is ambiguous. Noted under
*non-negotiables* below.

== Verdict

*Current CRG grade: **D (Alpha)**. Demote one grade from
self-declared C.*

Self-declared: *C*. Audited: *D*. One-grade honesty delta — smaller
than neurophone's two-grade delta (neurophone had committed
merge-conflict markers; proof-of-work does not) but still a demote.

Reasoning (one sentence): proof-of-work passes the D-floor cleanly
(workflows parse, SPDX is uniform, zero dangerous patterns, real
test code, `#![forbid(unsafe_code)]`) but fails the C clauses on
three independent axes — no `[dogfooding-status]` and no dogfood
log, per-suite test counts overclaim by 37-56% against grep, and the
authoritative `EXPLAINME.adoc` fabricates ten source paths and four
"dogfooded-across-the-account" integrations that the codebase does
not contain.

The D grade is earned, not conservative-by-default, on three positive
signals:

1. Real Rust code, cleanly structured: 5 294 LOC across 18 source
   files, `#![forbid(unsafe_code)]` at the library entry, zero
   `unsafe` blocks in `src/`, zero dangerous patterns in `src/`
   or `tests/`.
2. Real test surface: 79 `#[test]`/`#[tokio::test]` attributes
   across property / aspect / e2e / inline suites plus 5 `proptest!`
   blocks plus a criterion bench harness that was honestly
   de-fraudulented in `f9adea4`.
3. Clean structural hygiene: 27 workflows that parse, SPDX on 20/20
   sampled source files, LICENSES trident, `.well-known/`
   complete, fuzz + clusterfuzzlite scaffold present, zero
   committed merge-conflict markers.
